Guía de Facturación Electrónica (e-CF)

Este módulo permite activar e-CF por compañía sin afectar el flujo NCF tradicional.

  1. Activar FE por compañía y elegir modo.
  2. Emitir e-CF vía API /api/fe/emit.
  3. Programar cron cPanel: flask ecf_process_pending --limit 50.
  4. Monitorear estado con /api/fe/status/<doc_id>.

Modos disponibles: DIRECT_DGII, PSE_EXTERNAL, PSE_ECOSEA (stub).
